# Example Uses

## Example 1: Policy Research Digest

### Input

- Goal: 判断一份 AI 产业政策是否值得跟进
- Sources:
  - 政策征求意见稿
  - 官方公告页
  - 两篇相关新闻

### Expected Shape

1. 目标
2. 关键发现
3. 证据与来源
4. 含义判断
5. 待确认问题

### Sample Output Excerpt

## 1. 目标

- 判断这份政策对个人开发者的现实机会，而不是只看补贴上限。

## 2. 关键发现

- 政策重点支持 OpenClaw 生态中的技能包、场景化智能体和企业采购应用。
- 大额支持更偏企业项目，个人更现实的是先做可展示成果，再争取生态资源。

## 4. 含义判断

- 如果目标是提高审核通过率，项目应明显贴近 OpenClaw，而不是泛泛做 AI 工具。

## Example 2: Competitor Scan

### Input

- Goal: 比较三个 AI 内容工具的差异
- Sources:
  - 产品官网
  - 定价页
  - 用户评论摘录

### Output Notes

- 重点写差异，不要重复官网自述。
- 证据和判断必须分开。
