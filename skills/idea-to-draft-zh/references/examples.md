# Example Uses

## Example 1: Topic To Draft

### Input

- Topic: 为什么个人开发者应该先做 OpenClaw Skill，而不是一上来做大平台
- Audience: 想做 AI 副业的技术人
- Constraints:
  - 中文
  - 口语化但不空泛
  - 800 到 1200 字

### Expected Shape

1. 受众
2. 核心承诺
3. 角度备选
4. 最终结构
5. 初稿
6. 下一轮修改建议

### Sample Output Excerpt

## 2. 核心承诺

- 让读者理解，先做 Skill 更容易拿到真实反馈、真实用户和真实案例。

## 3. 角度备选

- 角度 A：从成本视角切入
- 角度 B：从政策审核友好度切入
- 角度 C：从个人执行难度切入

## Example 2: Notes To Draft

### Input

- Raw notes:
  - OpenClaw 还在早期
  - Skill 比平台轻
  - 更容易做演示
  - 政策第一条直接支持技能包

### Output Notes

- 先收束一个中心论点，再开始写。
- 不要把三四个角度混成一篇散文。
