# 用例示例

## 示例 1：生态周简报

### 输入

- 时间窗口：2026-03-04 至 2026-03-10
- 来源：
  - OpenClaw 文档更新
  - GitHub 问题列表
  - 一份政策页面
  - 两篇社区讨论

### 期望形态

1. 范围与时间
2. 本周发生了什么变化
3. 这些变化为什么重要
4. 下一步动作
5. 下周观察点

### 示例输出片段

## 2. 本周发生了什么变化

- 官方文档增加了新的技能说明页面，说明生态正在补齐开发者入口。
- 社区问题仍集中在部署和权限边界，说明安全与可用性仍是早期空白区。

## 4. 下一步动作

- 跑通 1 个真实技能演示，而不是继续停留在想法讨论。

## 示例 2：来源回顾

### 输入

- 来源：
  - 5 条新闻
  - 1 份周内产品更新记录

### 输出备注

- 只写本周新增变化。
- 没变化的旧信息不要占版面。
