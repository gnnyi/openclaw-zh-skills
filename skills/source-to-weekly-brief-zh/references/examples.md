# Example Uses

## Example 1: Weekly Ecosystem Brief

### Input

- Time window: 2026-03-04 to 2026-03-10
- Sources:
  - OpenClaw docs updates
  - GitHub issues
  - One policy page
  - Two community posts

### Expected Shape

1. 范围与时间
2. 本周发生了什么变化
3. 这些变化为什么重要
4. 下一步动作
5. 下周观察点

### Sample Output Excerpt

## 2. 本周发生了什么变化

- 官方文档增加了新的 Skill 说明页面，说明生态正在补齐开发者入口。
- 社区问题仍集中在部署和权限边界，说明安全与可用性仍是早期空白区。

## 4. 下一步动作

- 跑通 1 个真实 Skill demo，而不是继续停留在想法讨论。

## Example 2: Source Recap

### Input

- Sources:
  - 5 条新闻
  - 1 份周内产品更新记录

### Output Notes

- 只写本周新增变化。
- 没变化的老信息不要占版面。
